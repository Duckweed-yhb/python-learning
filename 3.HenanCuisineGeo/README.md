# HenanFoodMap - 河南美食地图可视化

## 项目简介
本项目基于 Python + GeoPandas + Matplotlib 实现河南省18个地市/示范区的特色美食地理可视化，支持：
- 精准绘制河南省行政区划地图，标注各地市特色美食
- 自定义调整文字位置、字体大小，优化视觉呈现效果
- 两种布局模式：右侧图例版 + 图片环绕地图版（放大紧凑版）
- 输出600DPI高清PNG和矢量SVG文件，可直接用于印刷/文创制作

## 核心功能
✅ 河南省18地市地理边界加载与渲染  
✅ 各地市特色美食文字标注（精准位置微调）  
✅ 美食图片可视化（统一尺寸+透明背景）  
✅ 张良镇特殊标记（红色高亮）  
✅ 高清图片导出（PNG/SVG双格式）  
✅ 中文显示兼容（解决Matplotlib中文乱码问题）
